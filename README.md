# Game Rental DBMS System

A fully functional **Game Rental Database Management System** with an **interactive Bash terminal interface** for managing users, games, copies, rentals, reviews, and waitlists.

# Features

- **Admin and User Management**: Create, update, and manage accounts for admins and users.
- **Game Catalog**: Store game details including title, developer, release year, description, genre, and age rating.
- **Game Copies**: Track available, borrowed, and reserved copies.
- **Borrows and Returns**: Record borrowings, due dates, returns, and overdue fees with automatic validation.
- **Reviews and Ratings**: Users can leave ratings (1–5) and comments for games.
- **Waitlist System**: Users can request games currently unavailable.
- **Interactive Bash Terminal**: Navigate the system through a command-line interface to perform CRUD operations.
- **Database Integrity**: Enforced via primary keys, foreign keys, unique constraints, and check constraints.

# Technologies Used

- **SQL / Oracle PL/SQL** – Database design, table creation, relationships, constraints, and sample data.
- **Bash (Shell Scripting)** – Interactive terminal interface to interact with the database.
- **SQL*Plus** – Executing SQL scripts and testing database functionality.
